⬆️ Dateien Hochladen

Wählen Sie die Dateien aus, die Sie hochladen möchten:

So erstellen Sie den Stammdaten-Export der Klassen:

  1. Gehen Sie in WebUntis zu Stammdaten → Klassen.
  2. Kopieren Sie die Tabelle in eine Excel-Datei.
  3. Die Datei sollte folgende Spalten in genau dieser Reihenfolge enthalten (nichts umbenennen): 
Auswahl, [eine Leere Spalte], Klasse, Langname, Alias, Jahrgangsstufe, Text, Klassenlehrkraft, Klassenlehrkraft, Abteilung, Von, Bis

Hinweis: Die Tabelle aus WebUntis kann einfach kopiert und eingefügt werden, ohne zusätzliche Änderungen vorzunehmen.

Exportieren Sie diese Excel-Datei anschließend mit Excel als .csv.

So erstellen Sie den Stammdaten-Export der Lehrkräfte:

  1. Gehen Sie in WebUntis zu Stammdaten → Lehrkräfte.
  2. Scrollen Sie nach unten und klicken Sie auf Berichte.
  3. Wählen Sie den CSV-Bericht bei "Lehrkräfte".
  4. **Wichtiger Hinweis:** Das Feld für die E-Mail-Adressen muss mit den Dienst-E-Mail-Adressen der Kollegen gefüllt sein, damit es nachher funktioniert.

So erstellen Sie den Schild NRW Export:

  • Auswahlsfilter in SchildNRW und Export:
  • Unten bei Laufbahninfo bei Schuljahr das aktuelle Schuljahr auswählen.
  • Oben rechts bei Status: Aktiv, Abschluss und Abgänger anwählen.
  • Speichern Sie diesen Filter, damit Sie ihn später über "Auswahl - Vorhandene Filter laden" wieder verwenden können.
  • Export aus SchildNRW als Text/Excel Export, jedoch unbedingt mit der manuell eingegebenen Dateiendung .csv.
  • Als Separator ist ; zu wählen.

Erforderliche Daten (in dieser Reihenfolge):

Interne ID-Nummer, Nachname, Vorname, Klasse, Klassenlehrer, Geburtsdatum, Geschlecht, vorrauss. Abschluss, Aufnahmedatum, Entlassdatum, Volljährig, Schulpflicht erfüllt, Status

Optionale Daten:

E-Mail (privat), Telefon-Nr., Fax-Nr., Straße, Postleitzahl, Ortsname

Hinweise:

  • Der Export funktioniert nicht, wenn die Datei als Excel-Datei exportiert und anschließend als .csv gespeichert wird.
  • Ergänzen Sie die Endung .csv manuell, nachdem Sie den Exporttyp Textdatei ausgewählt haben.
  • Speichern Sie die Exporteinstellungen als Vorlage ab, um sie später schneller verwenden zu können.
